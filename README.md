# TD.mp3
Standalone TouchDesigner app that works like a Windows media player from back in the day (with some twists)

![td-mp3-app](assets/img/ref/td-mp3-app.png)

# Media player features:
1. Open a folder
2. Audio file loads into player
3. Set your volume
4. Skip around the timeline
5. Play / Pause
6. Loop track
7. Adjust Speed (0.5x <- 1x -> 1.5x)

# Controls

Keyboard '4' | volume + 1
Keyboard '5' | volume = 0
Keyboard '6' | volume + 1

Keyboard '7' | track index - 1
Keyboard '8' | track index = 0
Keyboard '9' | track index + 1



# Visualizer

There are 2 audio-reactive visuals to choose from.
Click on the visual in the center of the GUI to cycle. <br>
<br>
A. RGB circle feedback vortex <br>
B. Audio specturm feeback wave
